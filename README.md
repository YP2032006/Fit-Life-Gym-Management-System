🏋️ FitLife Gym Management System
A full-stack Java web application for managing gym member records with secure authentication and real-time database integration. Built with JSP, JDBC, and IBM i (AS/400) to demonstrate enterprise-level web development.

🔧 Technologies Used

Java (JSP & Servlets)
JDBC
IBM i (AS/400)
Apache Tomcat
HTML5 & CSS3
NetBeans IDE

✨ Features

Secure session-based authentication
Real-time member data from IBM i database
Dynamic content rendering with JSP and JDBC
Responsive, mobile-friendly interface
Modular architecture with reusable JSP components
SQL injection protection with prepared statements

🚀 How to Run

Clone the repository and open in NetBeans
Add jt400.jar (IBM i JDBC driver) to project libraries
Configure Apache Tomcat server
Update database credentials in configuration
Deploy and access at http://localhost:8080/FitLife

💡 What I Learned
This project taught me enterprise Java web development—specifically session management, JDBC connectivity, and working with legacy database systems in real-world scenarios.

📌 Notes

Built as an academic project to demonstrate full-stack development skills
